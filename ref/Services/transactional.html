transactional

Purpose

Enables transaction demarcation if set to true

Examples

class BookService {
    static transactional = true
	…
}

Description

By default services do not have automatic transaction management such as propagation and so on. By setting transactional to true you get automatic Spring managed transaction demarcation for every method in the service. The propagation level of the transaction is by default set to PROPAGATION_REQUIRED.

See Declarative Transactions in the user guide for more information.
