elseif

Purpose

The logical elseif tag

Examples

<g:if test="${name == 'fred'}">
     Hello Fred!
</g:if>
<g:elseif test="${name == 'bob'}">
     Hello bob!
</g:elseif>
<g:elseif env="development">
     Hello unknown developer!
</g:elseif>

Description

Attributes

  • test - The expression to test
  • env - An environment name
